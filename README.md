# Advent of code

Simplement une résolution des exercices, proposés par le site [advent of code](https://adventofcode.com/), en Rust.
    
## Installer localement

Cloner le projet :

```bash
  git clone https://github.com/lucas-cosson/advent-of-code
```

Pour lancer ce projet sur une année particulière :

```bash
  cargo run -p year_2015
```
